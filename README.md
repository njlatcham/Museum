## Dependencies
Languages: HTML, Stylus, and JavaScript
Bundler: Parceljs

## Getting Started
1. Clone Repository
2. Install dependencies using 'npm install'
3. Start local server using 'parcel index.html'
4. Open browser to localhost:1234
